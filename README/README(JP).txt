【概要】


【注意事項 precautions 】
・動画解析をすべてローカルで行います
・TLデータを外部に送信することはありません
・通信機能を持ちません
・個人のプレイ検証を目的としたローカル解析ツールです
・TLの共有はユーザーの判断と責任で行われます
・Python 3.11 で動作確認しています
・3.12以降では一部ライブラリが動作しない場合があります

【使用方法】


tle(仮)はNexon GamesおよびYostarとは一切関係ありません。このツールで使用されるゲーム内の画像、情報、アセットの所有権および著作権はそれぞれの権利者に帰属します。

tl_extractor/
│
├── main.py           # アプリのエントリーポイント（起動用）
├── config.py         # 定数やハイパーパラメータ
├── utils.py          # 日本語パス対応、時間変換、YTダウンロードなど
├── state.py          # アプリケーション状態やスロット状態の管理
├── detector.py       # 画像処理・テンプレートマッチング・コスト検出コア
└── gui.py            # Tkinterによるメイン画面・ROI選択・切り取りUI
